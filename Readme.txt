制作一款塔防射击游戏：
1.用户界面为长方形，由上到下分为三部分：
   虫洞区：用户界面顶部20%为虫洞区，各种怪物由此批量生成，向下移动。
   塔防区：用户界面底部30%为塔防区，主人公位于距离塔防区上边界下方1 /3处，造型参考阿斯加德守护者Heimdall。
   攻防区：用户界面中间部分为攻防区，各种类型怪物向下移动，最多可以到达塔防区边界处。
2.游戏逻辑：
   主干逻辑：游戏开始后，各种怪物从虫洞区批量随机生成后，以一定速度经过攻防区向下方移动，如果怪物在到达塔防区上边界前未被消灭，即可对塔防区上边界进攻，消耗塔防区护甲值，护甲值归零则GameOver。游戏开始后，主人公对进入攻防区的怪物自动发射飞剑进行攻击，飞剑击中怪物消耗怪物生命值，怪物生命值归零则消失。
   用户操作逻辑：飞剑分为普通飞剑、寒冰飞剑、烈火飞剑。前200只怪物被消灭时，随机选择20只，触发飞剑选择提示框，
